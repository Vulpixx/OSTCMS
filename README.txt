OSTCMS is an OPEN SOURCE free content management system.

The Current version is BETA 0.0.6

Features:
- Easy Setup
- No Database
- Easy Configuration and Theme Change

Planned Features:
- Admin Control Panel
- Support for Images on Pages
- WYSIWYG Editor
- OSTCMS Software to make page editing much easier
- Templates
- Much More!

TO INSTALL:
1. Edit pages in 'Public (upload)' to your liking with your favourite text editor.
2. Upload to Your Website
3. Enjoy!
